IDEs :

An Integrated Development Environment (IDE) is a comprehensive software application that provides programmers with a suite of tools to write, test, and debug their code more efficiently. It combines several aspects of software development into one interface, including a source code editor, build automation tools, and a debugger.
I regularly use Visual Studio for the majority of my projects, taking full advantage of its integrated debugger to efficiently track down and fix bugs, enhancing the quality and reliability of my code. Visual Studio's robust environment supports my development workflow, making it a key tool in my programming arsenal.
For my Java-based projects, I turn to IntelliJ IDEA. This IDE is particularly beneficial due to its integration with Gradle, a powerful build automation system. Gradle streamlines the build process, manages dependencies, and automates tasks, making IntelliJ an invaluable resource for efficient and effective Java development.

R language :

R is a programming language and environment specifically designed for statistical computing and graphics. It is widely used among statisticians and data analysts for data mining, statistical analysis, and graphical representation of data. R is known for its extensive library of packages, offering a vast array of statistical techniques such as linear and nonlinear modeling, time-series analysis, classification, clustering, and others. It also excels in producing high-quality plots and graphs, which are essential for data visualization and interpretation.

QGIS :

QGIS is an open-source Geographic Information System (GIS) software, widely acclaimed for its ability to analyze, edit, and visualize geospatial information. It is a versatile tool for mapping and spatial data analysis, supporting various data formats and offering an intuitive interface. My practical involvement with QGIS includes the development of a bespoke plugin designed to serve the specific needs of municipal workers in Strasbourg. This plugin enabled field agents to perform their site surveys using a tablet, allowing them to report and centralize all information directly onto a map.
